What is DevOps Abbrevation?

DevOps stands for Devlopment and Operations

What is DevOps?

DevOps is a tools that is used to bring both operations and devlopment team together. The main goal is to shorten the time period of software development life cycle and to generate a quality software.

DevOps Tools:

There are various tools in DevOps. They are:

  1. Jenkins
  2. Git
  3. Ansible
  4. Docker
  5. Kubernets
  6. Prometheus
